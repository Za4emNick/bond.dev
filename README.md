# Bond.dev Portfolio

Этот репозиторий содержит статическую страницу портфолио. Автоматической кнопки для отправки правок в GitHub нет: нужно воспользоваться git.

## Как отправить изменения в GitHub
1. Проверить, что нужные файлы сохранены и добавлены: `git status` и `git add <files>`.
2. Зафиксировать коммит с понятным сообщением: `git commit -m "Update portfolio"`.
3. Отправить ветку на GitHub: `git push origin <branch>`.

Так правки из локального репозитория попадут в GitHub и смогут попасть в Pull Request или основную ветку после ревью.
